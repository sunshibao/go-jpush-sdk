# JPush GO SDK
极光推送golang sdk。
由于官方 暂时没有go sdk，故开发本SDK。
使用教程直接参照单元测试代码即可。

## 功能列表
+ Push API v3
+ Report API v3
+ Device API v3
+ Schedule API v3

## 单元测试
1. 添加**APP_KEY**和**MASTER_SECRET**以及**REGISTRATION_ID**到环境变量
2. go test

## 如何使用
参照单元测试即可
